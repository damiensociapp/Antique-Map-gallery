# Antique Maps Gallery

An ultra-modern, fully self-contained storefront concept for the antique-maps
niche — rare cartography from the 15th to the 19th century.

- **Single file, zero binary assets.** All cartographic artwork (planisphere,
  compass roses, wax-seal certificate, framed map engravings, continent icons)
  is hand-authored inline SVG, so the whole site is trivially replicable.
- **Palette:** parchment / ink / antique gold, with a cartographic teal accent.
- **Modern interactions:** scroll reveal, stat count-up, 3D card tilt, animated
  compass, mobile menu, `prefers-reduced-motion` support. Fully responsive.

## Run locally

Just open `index.html` in a browser — no build step, no dependencies
(fonts load from Google Fonts).

## Publish (GitHub Pages)

Settings → Pages → Source: **Deploy from a branch** → `main` / `/ (root)`.
The site is then live at `https://<user>.github.io/antique-map-gallery/`.
